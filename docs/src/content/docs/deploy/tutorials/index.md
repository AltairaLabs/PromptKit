Step-by-step tutorials for learning Deploy from basics to multi-environment workflows.

## Learning Path

These tutorials build progressively. Start from the beginning if you're new to Deploy.

### Beginner

1. [**First Deployment**](01-first-deployment) - Install an adapter, configure, and deploy (20 minutes)

### Intermediate

2. [**Multi-Environment**](02-multi-environment) - Dev, staging, and production environments (25 minutes)

## Prerequisites

Before starting these tutorials, ensure you have:

- PromptKit CLI (`promptarena`) installed
- A compiled `.pack.json` file (see [PackC tutorials](/packc/tutorials/))
- Basic command-line knowledge
- Access to a supported cloud provider (for applying changes)

## What You'll Learn

By completing these tutorials, you'll be able to:

- Install and manage deploy adapters
- Configure deployment targets in arena.yaml
- Preview changes before applying them
- Deploy packs to cloud providers
- Manage multiple environments from a single config
- Check deployment status and tear down resources

## Tutorial Format

Each tutorial includes:

- **Learning Objectives** - What you'll accomplish
- **Time Required** - Estimated completion time
- **Prerequisites** - What you need before starting
- **Step-by-Step Instructions** - Clear, actionable steps
- **Code Examples** - Complete, working examples
- **Verification** - How to confirm success
- **Next Steps** - Where to go from here
