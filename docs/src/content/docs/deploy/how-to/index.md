## By Use Case

### Development Workflow
- Install an adapter and test locally
- Plan changes before applying
- Destroy test environments when done

### Production Deployment
- Configure environment-specific settings
- Plan and review before applying
- Monitor status after deployment

### Team Collaboration
- Automate deployments in CI/CD
- Use environment configs for consistency
- Track state across team members

## Guide Format

Each how-to guide follows this structure:

1. **Goal** - What you'll accomplish
2. **Prerequisites** - What you need before starting
3. **Steps** - Specific actions to take
4. **Verification** - How to confirm success
5. **Troubleshooting** - Common issues and solutions
