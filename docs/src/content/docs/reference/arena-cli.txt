PromptKit Arena is a testing framework for running multi-turn conversation 
simulations across multiple LLM providers and system prompts.

It evaluates tone consistency, content quality, prompt adherence, and validates
conversation flows for various task types (customer support, code assistance, etc.).

Usage:
  promptarena [command]

Available Commands:
  completion     Generate shell completion script
  config-inspect Inspect and validate prompt arena configuration
  debug          Debug configuration and prompt loading
  help           Help about any command
  init           Initialize a new Arena test project
  mocks          Manage mock responses for PromptArena
  prompt-debug   Debug and test prompt generation
  render         Generate HTML report from existing results
  run            Run conversation simulations
  serve          Serve the output directory via HTTP for viewing reports with audio playback
  templates      Manage PromptArena templates (list, fetch, render)
  validate       Validate configuration files against JSON schemas
  view           Browse and view past Arena test results

Flags:
  -h, --help      help for promptarena
  -v, --version   version for promptarena

Use "promptarena [command] --help" for more information about a command.
